# ジャンル別レシピ — ギャラリー掲載傾向の分解

出典傾向: 国内の厳選デザインギャラリーに掲載される作品群の傾向を、デザイントークン（変数）として一般化したもの。
各レシピは 01-05 の原則に対する「変数の代入値」。原則と矛盾したらレシピを疑う。

## 共通の掲載水準（全ジャンル前提）

- 1pxのズレもない整列 / 一貫したトークン / レスポンシブ完全対応
- 写真・画像の品質がテキスト以上に印象を支配する（低品質画像は全設計を無効化する）
- 「らしさ」＝変数の一貫性。1ページ内でレシピを混ぜない

## A. Cool / テック / ブランド（依頼の主対象）

| 変数 | 値 |
|---|---|
| ベース色 | 黒 L0.13-0.18 またはダークネイビー、白抜き文字 L0.93 |
| アクセント | 1色のみ（電気的な青緑・ライム・オレンジ等、C 0.15+）比率3%以下 |
| ジャンプ率 | 4-8倍。巨大英字見出し（和文サイトでも見出しは英語が定番）＋負の字間 |
| レイアウト | ブロークングリッド/スプリット、非対称余白、画像とテキストの重なり |
| モーション | easeOutExpo、スクロール出現、ホバーで画像ズーム(scale 1.05, 600ms) |
| 質感 | 細い罫線(1px, 低コントラスト)、番号付きセクション(01/02/03)、余白大 |

失敗パターン: 黒背景＋派手色多数＝ゲーミング的安っぽさ。アクセント1色厳守。

## B. コーポレート / 信頼系（士業・医療・金融・製造）

- ベース: 白/オフホワイト、アクセント: 青系1色、補助にグレー
- ジャンプ率 1.5-2.5倍、明朝は理念ページのみ検討
- 対称レイアウト、カードグリッド、実績数値を大きく（数字はブランドより雄弁）
- モーション最小限（ホバー＋控えめフェードのみ）。奇抜さは信頼の敵

## C. LP / コンバージョン特化

- 1画面1メッセージ、ファーストビューに「誰の何をどうする」＋CTA
- CTA色はページ内で唯一の高彩度色（フォンレストルフ）、全セクション末に反復
- Z型骨格、社会的証明（実績・レビュー）を折返し直後に
- コントラスト・サイズは上品さより明瞭さを優先（Bと同じ変数でジャンプ率だけ3倍前後に上げる）

## D. ミニマル / ポートフォリオ / 建築・写真

- 色は白黒＋グレーのみ（アクセント0色も可）。作品画像が唯一の色
- タイポは小さめ・細め、余白最大級（セクション160px+）
- 余白による孤立が唯一の強調手段。罫線・影・装飾を全廃
- ホバーは opacity 0.6 か画像差し替えのみ

## E. ラグジュアリー / 和 / 伝統

- 明朝体（見出し）、縦書きの部分使用、金/生成り/深緑/墨のC低め配色
- ジャンプ率低（1.5-2倍）、字間広め(0.1-0.2em)、行間広め
- 遅めのモーション（600-800ms）、フェード主体。速い動きは安っぽい
- 写真は引きより寄り（質感・ディテール）

## F. 遊び / キッズ / エンタメ

- 高彩度3色（トライアド120°）、角丸大(16-24px)、太いウェイト
- バウンス系イージング解禁、イラスト・手書き要素
- 唯一「密度の高さ」が正義になりうるジャンル。ただし整列は守る

## ジャンル判定の質問（設計開始前に必ず答える）

1. ユーザーは誰で、サイトを見た3秒後にどんな感情になるべきか（形容詞3つで言語化）
2. 競合はどのレシピか。同じにするか、外すか（外すなら1変数だけ外す）
3. 最重要アクションは何か（=スコア1要素。1つに絞れるまで設計しない）
